# Snake Game для ESP32

## Опис гри

**Snake** - це класична аркадна гра, реалізована для мікроконтролера ESP32 з OLED-дисплеєм. Гравець керує змійкою, яка рухається по ігровому полю, їсть їжу для зростання та уникає зіткнень із стінами, перешкодами або власним хвостом. Мета - набрати максимальну кількість очок.

### Основні особливості
- **Ігрове поле**: 32x8 клітинок (128x64 пікселів дисплея).
- **Змійка**: Рухається в чотирьох напрямках (вверх, вниз, ліворуч, праворуч).
- **Їжа**: З'являється випадково на полі, блимає для візуального ефекту.
- **Перешкоди**: Випадкові перешкоди на полі для ускладнення гри.
- **Рекорд**: Зберігається в енергонезалежній пам'яті ESP32.
- **Стани гри**:
  - **SPLASH**: Початковий екран з анімацією заголовка та рекорду.
  - **PLAYING**: Основна гра.
  - **COLLISION**: Анімація зіткнення (блимання змійки).
  - **GAME_OVER**: Екран кінця гри з фінальним рахунком.
- **Керування**:
  - Ліва кнопка (GPIO 4): Поворот змійки ліворуч.
  - Права кнопка (GPIO 18): Поворот змійки праворуч.
  - Початок гри: Натиснути будь-яку кнопку на сплеші та відпустити.
  - Скидання рекорду: Утримувати обидві кнопки 2 секунди на сплеші.
  - Перехід після смерті: Натиснути кнопку на екрані Game Over для повернення на сплеш.
- **Візуальні ефекти**:
  - Анімація сплешу.
  - Блимання їжі.
  - Блимання змійки при зіткненні.
  - Центрування елементів на екрані.

### Як грати
1. Після запуску ESP32 з'являється сплеш-екран з анімацією "SNAKE" та поточним рекордом.
2. Натисніть будь-яку кнопку та відпустіть, щоб почати гру.
3. Керуйте змійкою: ліва кнопка - поворот ліворуч, права - праворуч.
4. Змійка автоматично рухається вперед; їжте їжу (білі квадрати), щоб рости.
5. Уникайте стін, перешкод та власного хвоста.
6. При зіткненні гра переходить у стан COLLISION з блиманням, потім GAME_OVER.
7. На GAME_OVER натисніть кнопку, щоб повернутися на сплеш.
8. Для скидання рекорду на сплеші утримуйте обидві кнопки 2 секунди - гра почнеться відразу після скидання.

### Технічні деталі
- **Платформа**: ESP32 (DOIT ESP32 DEVKIT V1).
- **Дисплей**: SSD1306 OLED 128x64 пікселів (I2C).
- **Бібліотеки**:
  - Adafruit GFX та SSD1306 для графіки.
  - ezButton для обробки кнопок.
  - Preferences для збереження даних.
- **Швидкість**: Змійка рухається з інтервалом 300 мс (змінюється залежно від рівня).
- **Пам'ять**: Рекорд зберігається у NVS ESP32.

## Структура коду

Проект побудований на Arduino Framework з використанням PlatformIO. Код модульний, розділений на файли для кращої організації.

### Загальна архітектура
- **Клас SnakeGame**: Основний клас, що містить всю логіку гри.
- **Станова машина**: Гра працює у чотирьох станах, перемикання між якими відбувається в методі `run()`.
- **Модулі**:
  - `Game.cpp`: Обробка станів та введення.
  - `GameLogic.cpp`: Ігрова логіка (рух, їжа, скидання).
  - `GameRender.cpp`: Малювання екранів.

### Файли проекту

#### platformio.ini
Конфігураційний файл PlatformIO.
- Визначає плату ESP32.
- Налаштовує залежності: Adafruit SSD1306, ezButton, Preferences.
- Команди збірки та завантаження.

#### include/Game.h
Заголовний файл класу SnakeGame.
- **Константи**:
  - `SCREEN_WIDTH = 128`, `SCREEN_HEIGHT = 64`: Розмір дисплея.
  - `CELL_SIZE = 4`: Розмір клітинки (пікселів).
  - `GRID_WIDTH = 32`, `GRID_HEIGHT = 14`: Розмір поля в клітинках.
  - `BUTTON_LEFT_PIN = 4`, `BUTTON_RIGHT_PIN = 18`: Піни кнопок.
  - `LONG_PRESS_DURATION_MS = 2000`: Час для довгого натискання (скидання рекорду).
  - `COLLISION_BLINK_DURATION_MS = 2000`: Тривалість екрана зіткнення.
- **Структури**:
  - `Point`: Координати (x, y).
- **Переліки**:
  - `GameState`: SPLASH, PLAYING, COLLISION, GAME_OVER.
- **Змінні стану**:
  - `snake[200]`: Масив точок змійки.
  - `food`: Координати їжі.
  - `obstacles[MAX_OBSTACLES]`: Перешкоди.
  - `snakeLength`, `dir`: Довжина та напрямок змійки.
  - `highScore`: Рекорд.
  - `gameState`: Поточний стан.
  - Прапорці: `foodOn`, `showReleaseMessage`, `wasLongPress` тощо.
- **Методи**:
  - Публічні: `SnakeGame()`, `begin()`, `run()`.
  - Приватні: Методи логіки та малювання.

#### src/main.cpp
Точка входу Arduino.
- Створює екземпляр `SnakeGame`.
- Викликає `begin()` в `setup()`.
- Викликає `run()` в `loop()`.

#### src/Game.cpp
Основна логіка станів та введення.
- **Конструктор**: Ініціалізує змінні, налаштовує кнопки.
- **begin()**: Ініціалізує дисплей, завантажує рекорд.
- **updateInput()**: Оновлює стани кнопок на кожному тіку циклу.
- **run()**: Головний цикл, switch по станам.
  - **SPLASH**:
    - Малює сплеш.
    - Обробляє натискання: на відпускання починає гру, на довге натискання обох кнопок скидає рекорд.
    - Показує повідомлення "Release to start" та "Hold both for reset".
  - **PLAYING**:
    - Обробляє повороти (тільки один на крок).
    - Рухає змійку кожні `moveDelay` мс.
    - Перевіряє зіткнення.
  - **COLLISION**:
    - Малює блимання змійки.
    - Після часу переходить на GAME_OVER, зберігає рекорд.
  - **GAME_OVER**:
    - Малює екран кінця.
    - На натискання переходить на SPLASH.
- **Методи введення**: Використовує `isPressed()` для подій (повороти), `getState()` для станів (сплеш).

#### src/GameLogic.cpp
Ігрова логіка.
- **spawnFood()**: Генерує їжу у випадковому місці, не на змійці чи перешкодах.
- **resetGame()**: Скидає змійку, напрямок, швидкість і випадково розставляє перешкоди.
- **turnLeft() / turnRight()**: Змінює напрямок, якщо не щойно повернув.
- **moveSnake()**: Рухає голову, перевіряє їжу/зіткнення, оновлює хвіст.

#### src/GameRender.cpp
Малювання екранів.
- **drawSplash()**: Анімація заголовка та рекорду, або повідомлення залежно від `showReleaseMessage`.
- **drawGame()**: Малює поле, змійку, їжу (блимає), перешкоди, рахунок.
- **drawCollision()**: Почергово показує кадр гри та порожній екран для ефекту блимання.
- **drawGameOver()**: Анімований екран "Game Over" з поточним рахунком.

### Ключові механізми
- **Обробка введення**: ezButton з debounce. `isPressed()` для одноразових подій, `getState()` для тривалого стану.
- **Таймінг**: `millis()` для рухів, блимання, анімацій.
- **Збереження**: Preferences для рекорду.
- **Випадковість**: `randomSeed(micros())` для їжі та перешкод.
- **Модульність**: Код розділений для легкого обслуговування.

### Збірка та запуск
1. Встановіть PlatformIO.
2. Відкрийте проект у VS Code з PlatformIO.
3. Підключіть ESP32.
4. Запустіть `pio run --target upload`.
5. Гра запуститься автоматично.
